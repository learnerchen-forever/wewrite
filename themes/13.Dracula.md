# 1. 风格特点

```CSS
:root {
	--wewrite-primary: #BD93F9; /* Dracula: 紫色 - 用于强调或主要元素 */
 	--wewrite-secondary: #FF79C6; /* Dracula: 粉色 - 用于次要强调或装饰 */
 	--wewrite-text: #F8F8F2; /* Dracula: 前景文本色 */
}
body, p, span, li {
    font-size: 1em; /* 手机公众号基础字体通常是 1em，相当于 16px 或 17px 左右 */
    line-height: 1.8; /* 行高建议稍微大一点，增加阅读舒适度 */
}
/* 让 figure 标签居中 */
figure {
    display: block !important; /* 强制 figure 为块级元素 */
    margin: 1.5rem auto !important; /* 上下边距 1.5rem，左右自动居中 */
    /* 移除 figure 自身可能的内联 text-align: center，因为我们希望 figure 整体居中 */
    text-align: center !important; /* 保留 figure 内部的 text-align: center, 确保图片作为行内元素也能居中 */
}

/* 保持 img 自身的样式，但移除可能冲突的 margin: auto */
img {
    max-width: 100% !important;
    height: auto !important;
    display: block !important; /* 保持图片为块级元素 */
    margin: 0 auto !important; /* 这里我们让图片在 figure 内部居中 */
}
```
